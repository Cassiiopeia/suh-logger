# Contributing

1. Fork this repository
2. 새 브랜치 생성: `git checkout -b feature/your-feature`
3. 변경 사항 커밋: `git commit -m "Add some feature"`
4. 푸시: `git push origin feature/your-feature`
5. Pull Request 생성

## 코드 스타일
- Java 17, Lombok 사용
- 메서드별 Javadoc 필수
- 테스트 코드 작성 (JUnit 5)

## 리뷰 프로세스
- PR 에서 변경된 모듈별로 리뷰어 지정
- CI 통과 후 머지
